# Auto-Shroom-Software

Computer Vision Automation Bot – Independent Project Designed
and developed a large-scale Python automation platform focused on
real-time visual detection, event-driven actions, and system reliability.
Built custom computer vision and ML pipelines using YOLO,
TensorFlow, OpenCV, and NumPy, including multiple models for
visual classification and response handling. Integrated Arduino-based
hardware input control for stable device-level interaction, and
managed a subscription licensing.

The production repository will remain private because the application is still being sold.
The website contains videos showing the actual capabilities of the application.
